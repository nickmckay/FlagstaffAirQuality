About this map

Animated particulate matter (PM1.0, PM2.5, PM10) concentrations around Flagstaff, Arizona, combining two sensor networks: PurpleAir and Air Quality Egg. Data refresh every 15 minutes via GitHub Actions.

Calibration

PurpleAir PM2.5 is corrected with the EPA correction equation (Barkjohn et al. 2021), which compensates for the sensors' known overestimation using relative humidity. All PurpleAir values are then mapped onto the Air Quality Egg scale with a locally fitted robust regression, refit weekly from co-located hourly averages (applied only when the fit is statistically sound).

Reading the map

Circles are sensors (rings mark Air Quality Eggs; plain circles are PurpleAir). The shaded surface is an ordinary-kriging interpolation using an exponential variogram refit weekly from the archive, drawn over a rectangle enclosing the sensor network plus a 5 km buffer. Between sensors the surface is a statistical estimate, not a measurement. All three pollutants use the EPA AQI category colors; PM1.0 has no official breakpoints, so the PM2.5 breakpoints are applied. Values are µg/m³, times are local (Arizona).
